
# Unary Calculator Turing Machine

This repository contains my CS154 Unary Calculator term project files implemented using JFLAP 7.1.

## Project Overview

The Unary Calculator is a programmable calculator that works with unary numbers. It uses a Turing Machine tape to simulate Display, Input String, and Memory.

## Versions

- v4.0: Fixed bugs in the provided UC v3.0 implementation.
- v4.1: Added the SW instruction to swap Display and Memory.
- v4.2: Added the PM instruction to multiply Display by Memory and store the result in Display.

## Tools Used

- JFLAP 7.1
- Single-tape Turing Machine
- Turing Machine with Building Blocks
- Transducer mode
